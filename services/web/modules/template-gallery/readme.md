# Template

We express sincere gratitude to yu-i-i/overleaf-cep for their invaluable contributions to the development of this module.

This module provides the Template Gallery feature in Overleaf, enabling users to create new projects from a variety of templates. It includes functionalities for managing templates, rendering template selection views, and handling project creation based on selected templates.
